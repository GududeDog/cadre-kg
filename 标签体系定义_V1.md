## 一、标签体系总览

```
标签体系
├── 干部标签（5大类12维度，~270个标签值）
│   ├── 1. 评价标签 ── 4维度 ~62个标签
│   ├── 2. 熟悉领域 ── 1维度 ~50个标签
│   ├── 3. 重要经历 ── ~10子类 ~60个标签
│   ├── 4. 重要职务 ── ~12子类 ~75个标签
│   └── 5. 其他标签 ── 5子类
│
├── 班子标签（8大类 ~35个指标）
│   ├── 班子概览 / 班子预警 / 班子运行情况
│   ├── 班子结构分析 / 历史沿革 / 优秀年轻干部
│   └── 监督信息 / 存在问题及优化措施
│
└── 岗位标签（5大类）
    ├── 岗位基本信息 / 岗位历任干部来源去向
    ├── 历任干部信息 / 历任干部结构信息
    └── 岗位能力要求
```

#### 2.1.5 评价标签展示与计算规则

1. **多材料去重**：标签按材料时间倒序排列，同一年份完全相同的内容合并展示
2. **高频/低频展示**：优先展示个人高频词、全量低频词，词云不超过10个
3. **原值存储**：标签值 + 原始值双存，画像页面展示原值，查询使用标签值
4. **控词约束**：LLM 抽取时只能从清单中选择，不得自行新增标签

### 2.5 其他标签

#### 2.5.1 主要不足

展示方式：按年度倒序，同一年份相同内容合并展示，内容按材料顺序列举。

示例：
```
2023年：1. 学习理论不够系统
        2. 创新意识不够强
2022年：工作方式需改进
```

## 七、展示优先级规则

1. **材料权重排序**：考察材料(0.95) > 巡察报告(0.95) > 任免表(0.90) > 民主测评(0.90) > 述职报告(0.75) > 普通工作总结(0.60)
2. **时间倒序**：同权重材料按时间倒序
3. **高频/低频控制**：个人高频词优先，全量低频词次之，控制词云不超过10个
4. **去重合并**：同一年份完全相同的内容合并展示
